"# Quiz Java

simple console-based quiz  in java

## what it does
- asks 5 programming questions
- shows options A B C D
- calculates your score at the end

## how to run
```
javac Main.java
java Main
```

## files
- Main.java - starts the quiz
- Question.java - question class with getters/setters
- QuestionService.java - has all questions and scoring logic"
